Condensing the raw data to a more useful condensed version
====

Json data that contains information on motions. This includes:

<ol>
    <li>_id</li>
    <li>date</li>
    <li>Region</li>
    <li>Country</li>
    <li>City</li>
    <li>Tournament</li>
    <li>Round</li>
    <li>Motion</li>
    <li>Untranslated Motion</li>
    <li>Infoslide</li>
    <li>Untranslated Infoslide</li>
    <li>Level</li>
    <li>Types</li>
    <li>Likes</li>
    <li>Dislikes</li>
    <li>URL</li>
    <li>Pro Arguments</li>
    <li>Con Arguments</li>
    <li>Complexity</li>
    <li>Slug</li>
    <li>Video URLs</li>
    <li>__v</li>
    <li>Created At</li>
</ol>
